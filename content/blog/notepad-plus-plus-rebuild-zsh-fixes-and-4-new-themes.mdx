Hey vampires 👋🏻

We have some _behind-the-scenes_ updates on what we have been building across Dracula and Dracula Pro.

## zsh

We fixed several **zsh** bugs, including a tricky one where `$DRACULA_ARROW_ICON` could be interpreted as a CLI flag when its value started with `-`.

This is now handled correctly with `print -P --`.

```zsh
❯ print -P "--> "
print: bad option: ->
❯ print -P -- "--> "
-->
```

## Notepad++ (Open Source & Pro)

We rebuilt the [**Notepad++**](/notepad-plus-plus) theme **from scratch**.

Instead of maintaining XML manually, we now use a TypeScript build system that generates it programmatically. Contributors can [edit a theme definition](https://github.com/dracula/notepad-plus-plus/blob/main/themes/theme-types.ts) and simply run `npm run build`.

On top of that, we shipped:

- Full language coverage with **14 previously missing lexers**.
- Colors aligned with the official Dracula specification.
- Font settings removed from the XML, so users can configure fonts themselves.
- Fixes for the **Rust lexer** and the current line color.

## New on Dracula

This week, 4 new themes have joined the Dracula clan:

- [**MacDown**](/macdown-css): an open-source **Markdown editor** for macOS. This one is for the rendered Markdown preview.
- [**Markdown CSS**](/markdown-css): a polished **stylesheet** for Markdown documents.
- [**Dracula CSS**](/dracula-css): a reusable **CSS theme foundation** for Dracula styling.
- [**Signal Desktop**](/signal-desktop): a secure **messaging app** for desktop.

> Big thanks to [@jamesfishwick](https://github.com/jamesfishwick) for the three great additions.
