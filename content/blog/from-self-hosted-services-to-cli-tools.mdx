## New on Dracula

We have 9 new additions to the Dracula clan. From self-hosted services to CLI tools, there is something for everyone.

- [Unraid](/unraid): **NAS and app server OS.** A NAS and app server OS for self-hosting.
- [DankMaterialShell](/dankmaterialshell): **Wayland desktop shell.** A Wayland desktop shell for Linux.
- [Wiremix](/wiremix): **PipeWire audio mixer.** A PipeWire audio mixer from the terminal.
- [Snappy Driver Installer Origin](/snappy-driver-installer-origin): **Driver updater for Windows.** An open-source driver updater for Windows.
- [Superset](/superset): **Gokrazy monitoring dashboard.** A monitoring dashboard for Gokrazy systems.
- [Miniflux](/miniflux): **Self-hosted RSS reader.** A minimal, self-hosted RSS reader.
- [Dash](/dash): **Python web apps.** Python web apps for data and analytics.
- [Delta](/delta): **Syntax-highlighted git diffs.** Syntax-highlighted git diffs in your terminal.
- [Taskwarrior](/taskwarrior): **CLI task manager.** A CLI task manager for your workflow.
