Cursor is taking over with its AI features.

While Dracula was already available through the VSCode theme, we’ve crafted a custom-made Dracula theme just for Cursor.

[![Cursor Screenshot](/images/blog/cursor-theme-five-new-additions-and-more-a.webp)](/cursor)

## New on Dracula

We have five new additions to the clan:

- [Anytype](/anytype): **Local-first, privacy-focused note-taking app.** A local-first, privacy-focused note-taking app- [Nilesoft](/nilesoft-shell): a extensible Windows shell extension framework- [Snappy Driver Installer](/snappy-driver-installer): an open-source driver updater for Windows systems- [TablePlus](/tableplus): a native database management tool with a clean interface- [Cursor](/cursor): the AI Code Editor.
