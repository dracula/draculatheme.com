This update adds a new Alacritty theme, JetBrains EAP support, and additional desktop icons.

### Alacritty

Added Dracula Pro theme for Alacritty terminal emulator.

![Dracula Pro for Alacritty](/images/logs/new-alacritty-theme-and-jetbrains-update-a.png)

Added support for [_EAP 2020.3_](https://blog.jetbrains.com/idea/2020/09/intellij-idea-2020-3-eap/) (Early Access Program).

![WebStorm 2020.3 EAP](/images/logs/new-alacritty-theme-and-jetbrains-update-b.png)

### Desktop Icons

Added desktop icons for Alacritty and Android Studio.
