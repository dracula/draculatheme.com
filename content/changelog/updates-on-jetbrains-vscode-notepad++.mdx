### Highlights

This update brings improvements across JetBrains, VSCode, and Notepad++.

### JetBrains

- Added support for _EAP 2020.2_ (Early Access Program);
- Standardized syntax colors for _HTML_, _XML_, _CSS_, _JS_, _JSON_, _Go_, _Ruby_, _Python_, _PHP_, _Java_, and _C#_.

### VSCode

- Enabled semantic highlighting;
- Fixed terminal bug by modifying hover background color;
- Enhanced whitespace color;
- Matched breadcrumb color to active tab color;
- Added support for editor.foldBackground highlight group.

### Notepad++

- Added support for LaTeX highlighting;
- Included note about Markdown syntax.
