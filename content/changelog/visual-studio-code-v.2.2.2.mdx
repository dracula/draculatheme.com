### Highlights

This update brings various fixes, adjustments, and improvements to Visual Studio Code and its forks (Cursor, Windsurf, etc.).

### Key updates

- Refined theme metadata, including improvements to extension presentation, descriptive information, and iconography;
- Updated license for compliance with general package guidelines;
- Fixed minimap system issues;
- Enhanced active editor indent guide color for better visibility;
- Corrected background colors for inserted text, eliminating opacity issues that were hiding underlying decorations;
- Improved debugger colors and related status indicators;
- Adjusted user interface element borders;
- Fixed PHP string quote coloring and namespace separator corrections.
