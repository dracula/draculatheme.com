JetBrains

This theme works with all JetBrains IDEs like IntelliJ, WebStorm, PyCharm, PhpStorm, etc.

Whenever you see a {NameOfIDE}, translate it to the IDE you're using.

Whenever you see a {You}, translate it to your username on your local machine.

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

$ git clone https://github.com/dracula/jetbrains.git

Install manually

Download using the GitHub .zip download option and unzip them

Activating theme

  1. Windows
    1. Create a folder called colors inside C:/Users/{you}/.{NameOfIDE}/config/colors/.
    2. Copy the ./jetbrains/Dracula.icls file to the C:/Users/{you}/.{NameOfIDE}/config/colors/ directory on your local machine.
  2. Mac/Linux
    1. Create a folder called colors inside ~/.{NameOfIDE}/config/colors/.
    2. Copy the ./jetbrains/Dracula.icls file to the ~/.{NameOfIDE}/config/colors/ directory on your local machine.
  4. Close and re-open {NameOfIDE}.
  5. Close and re-open {NameOfIDE}.
  6. Navigate to {NameOfIDE} -> Preferences -> Editor -> Colors & Fonts.
  7. Select “Dracula” from the dropdown menu.
  8. Click the “Ok” button to activate the color scheme.
