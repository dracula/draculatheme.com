Dracula

A dark theme for Zsh and 89+ apps

Zsh

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo into your oh-my-zsh folder with the following commands:

$ cd $ZSH_CUSTOM/themes 
$ git clone https://github.com/dracula/zsh.git dracula

Activating theme

Go to your ~/.zshrc file and set ZSH_THEME="dracula/dracula".

Install using zplug

Just add zplug "dracula/zsh", as:theme to your ~/.zshrc file.
